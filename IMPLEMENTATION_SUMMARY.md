# GCOS虚拟机实现总结

## 项目概述

本项目基于《卡及身份识别安全设备片上操作系统第3部分：支持面向过程应用后下载的基础层技术要求》(GB/T 44901.3)标准，实现了一个完整的智能卡虚拟机系统。

## 已完成的工作

### 1. 核心架构设计 ✅

#### 1.1 类型系统 (`vm_types.h`)
- 基本数据类型定义 (u8, s8, u16, s16, u32, s32等)
- 常量定义（栈大小、堆大小、文件类型标识符等）
- 枚举类型（VM状态、模块类型、数据类型、异常类型等）
- 核心数据结构（AID、栈帧、函数头、模块、应用实例、事务上下文等）

#### 1.2 虚拟机核心 (`vm_core.h/c`)
- VMContext结构：包含所有运行时状态
- 完整的生命周期管理（create/destroy/init/reset）
- 应用管理（select/deselect/install/uninstall）
- APDU命令执行接口
- 事务管理（begin/commit/rollback）
- 统计信息收集

#### 1.3 内存管理 (`vm_memory.h/c`)
- **执行器栈**：4字节单元，支持push/pop/peek操作
- **间接访问变量栈**：16字节单元，用于组合数据类型
- **全局数据区**：存储模块全局数据和临时数据
- **堆管理器**：简单的线性分配器，支持alloc/free/realloc
- **模块数据管理**：全局数据、只读数据、域数据的分配和回收
- **应用数据管理**：6种数据类型的创建和管理
  - 临时静态数据
  - 临时动态数据
  - 跨域数据
  - 持久性数据
  - 应用域数据
  - 引用域数据
- 内存保护机制

#### 1.4 指令集 (`vm_instructions.h/c`)
实现了200+条指令，包括：

**控制流指令** (0x00-0x2F)
- NOP, TRAP, RET, IRET
- BR.S8, BR.S16（无条件跳转）
- BEQZ, BNEZ, BEQ, BNE（条件跳转）
- CALL.U16, CALLIND, CALLEX（调用指令）

**栈操作** (0x30-0x3F)
- POP, POP.IS, PICK

**常量加载** (0x40-0x5F)
- LDC.0 ~ LDC.8（常用常数）
- LDC.M1（-1）
- LDC.U8, LDC.U16, LDC.I32
- LDC.A8, LDC.A16（地址加载）

**算术运算** (0x60-0x7F)
- ADD, SUB, MUL, DIVS, DIVU
- REMS, REMU
- 紧凑格式指令（ADD.T2_C6等）

**逻辑运算** (0x80-0x9F)
- AND, OR, XOR
- SHL, SHRS, SHRU
- ROTL, ROTR

**类型转换** (0xA0-0xAF)
- CVT.U8, CVT.U16

**比较指令** (0xB0-0xBF)
- EQZ, EQ, NE
- LTS, LTU, GTS, GTU, LES, LEU, GES, GEU

**数据访问** (0xD0-0xEF)
- LDT.0~8, STT.0~8（局部变量访问）
- STLDT.0~8（全局/域变量访问）
- LDT, STT（索引访问）

**内存块操作** (0xF0-0xFF)
- LDMS8, LDMU8, LDMS16, LDMU16, LDM32
- STM8, STM16, STM32
- MCOPY, MFILL

**异常处理**
- TRY, CATCH

每条指令都包含：
- 操作码定义
- 助记符
- 指令分类
- 操作数类型
- 详细描述

#### 1.6 文件加载器 (`vm_loader.h/c`)

## 使用示例

```c
#include "vm_core.h"

int main() {
    // 创建虚拟机
    VMContext *vm = vm_create();
    
    // 加载SEF文件
    vm_load_file(vm, "app.sef");
    
    // 选择应用
    AID aid = {...};
    vm_select_app(vm, &aid, 0);
    
    // 执行命令
    u8 apdu[] = {...};
    u8 response[256];
    u32 resp_len = sizeof(response);
    vm_execute_apdu(vm, apdu, sizeof(apdu), response, &resp_len);
    
    // 清理
    vm_destroy(vm);
    return 0;
}
```
